<code>@SpringBootApplication</code> - аннотация в SpringBoot, использующаяся для авто-конфигурации, поиска компонентов и добавления дополнительных конфигураций в классе приложения.

**Эквивалентные аннотации:**

<code>@Configuration</code> - содержит настройки конфигурации.

<code>@ComponentScan</code> - ищет все классы с аннотациями и регистрирует их как компоненты.

<code>@EnableAutoConfiguration</code> - автоматически настраивает SpringBoot на основе добавленных зависимостей.


**Параметры:**

<code>scanBasePackages</code> - позволяет указать пакеты, которые Spring Boot должен сканировать для поиска компонентов.

<code>scanBasePackageClasses</code> - позволяет указать классы, пакеты которых нужно сканировать.

<code>exclude</code> - позволяет исключить определенные автоматические конфигурации.

<code>excludeName</code> - позволяет исключить определенные автоматические конфигурации по названию класса.

<code>proxyBeanMethods</code> - управляет созданием прокси для бинов в конфигурационных классах.
